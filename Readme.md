 🚖 SevenTrip

Application de réservation de taxis et de location de véhicules.

Technologies utilisées
- HTML / CSS / JavaScript
- Git & GitHub

 Pages du site
- Accueil
- Connexion / Inscription
- Réservation de taxi
- Location de véhicule
- Historique (à venir)

Installation
1. Clone le dépôt : `git clone https://github.com/ton-pseudo/seven-trip.git`
2. Ouvre `index.html` dans ton navigateur

Prochaines étapes
- Ajout de JavaScript pour les interactions
- Connexion à une base de données
- Version mobile (responsive)
